\mysection{Exercise 1}
Consider the following finite difference approximations
\begin{enumerate}
    \item First order forward difference approximation of the first derivative
    of $u$
    \begin{equation}
        \label{eq:o1_d1_forward}
        D_1 u(x) = \frac{u(x+\Delta x) - u(x)}{\Delta x}
    \end{equation} 
    \item Second order central difference for second order derivative of $u$
     \begin{equation*}
        D_2 u(x) = \frac{u(x+\Delta x) - 2 u(x) + u(x-\Delta x)}{\Delta x^2}
     \end{equation*}
    \item Second order central difference for multi-variate function 
\end{enumerate}
Analyze the order of accuracy of the above approximation schemes. Compare the
accuracy of \eqref{eq:o1_d1_forward} with the center difference scheme (?? in
lecture notes).
